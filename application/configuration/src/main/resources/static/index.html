Episodie HTTP RESTful API

Fast and clean REST API that's easy to use, returns JSON.
It provides various data about tv shows.

The root url is https://episodie.ct8.pl/api and the available endpoints are documented at https://github.com/hypeapps/episodie-api
This is open source project. If you have any questions or you would contribute to the project, create pull request on GitHub project page.
